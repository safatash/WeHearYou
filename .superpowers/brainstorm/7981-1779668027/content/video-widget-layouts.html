Architecture: new VideoTestimonialWidget model (separate from ReviewWidget — different data source)
  • Fields: id, publicToken, organizationId, locationId, layout, isActive, createdAt
  • Public API: /api/public/video-widgets/[token] — returns published testimonials for that location
  • Embed script: extend existing /embed/widget.js with a video render mode, or separate /embed/video-widget.js
  • Widget creation: /widgets/new gets a type picker — "Review Widget" vs "Video Testimonials Widget"
  • Clicking a video card in the embedded widget opens a lightbox/modal player — no page navigation
