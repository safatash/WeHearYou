How should video testimonial requests fit into the existing send flow?

You already send review requests via email/SMS to contacts. The question is how video testimonial requests layer on top of that.

A

Request Type Picker at "Send New Request"

The existing "Send New Request" screen gets a type toggle at the top: Review Request or Video Testimonial Request. Same contacts list, same channels. Two separate sends, one shared entry point.

B

Send Both from One Compose Screen

One send flow where you can check Review Request and/or Video Testimonial. One contact, one send, two messages (or one combined message with both links). Campaigns list shows both types together.

Contact: Jane Smith
✓ Send Review Request
✓ Send Video Testimonial Request
Send →
C

Video Testimonial as a Follow-up Request

Review requests stay as-is. After a contact responds to a review, you get a prompt in the app: "Want to request a video testimonial from Jane?" — turning satisfied reviewers into video testimonial candidates naturally.
