NEW LIB FILES
src/lib/gbp-api.ts
Thin wrapper around the GBP REST API. Functions: publishReply, createPost, deletePost, uploadPhoto, deletePhoto, listPhotos, listQuestions, answerQuestion. Takes accessToken, handles refresh via existing GoogleAccountConnection.
src/lib/gbp-scheduler.ts
Called by the cron route. Queries GbpPost/GbpPhoto where status = SCHEDULED AND scheduledAt <= now. Calls gbp-api.ts for each. Updates status to PUBLISHED or FAILED.
src/lib/gbp-sync.ts
Nightly Q&A sync. Calls listQuestions, upserts GbpQuestion rows, detects new unanswered ones, fires team notification email.
src/app/api/cron/gbp/route.ts
Vercel cron endpoint. Secured with CRON_SECRET header check. Calls gbp-scheduler.ts (every 5 min) and gbp-sync.ts (nightly via separate schedule in vercel.json).
vercel.json crons: "schedule": "*/5 * * * *" → gbp-scheduler (posts + photos)    schedule": "0 2 * * *" → gbp-sync (Q&A nightly at 2am)
